Get the key detail and the block number of last modification of the given DID. It assumes that the DID has only 1 public key which is true for now but will change later. This function will then be modified to indicate which key(s) of the DID should be used. If DID is not registered an error is raised.

pub fn verify_sig_from_did(

    message: &[u8],
    did: &Did
) -> Result<bool, DispatchError>[src]

Verify given signature on the given message with the given DID's only public key. It assumes that the DID has only 1 public key which is true for now but will change later. This function will then be modified to indicate which key(s) of the DID should be used. If DID is not registered an error is raised. This function is intended to be used by other modules as well to check the signature from a DID.
